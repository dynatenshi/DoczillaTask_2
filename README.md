# Тестовые задания в компанию DOCZILLA

В данном репозитории содержится source-код задания.

Для компиляции и запуска требуется запустить файл **_START.bat**.

Альтернативый вариант: 
1. В консоли установить директорию на основную папку задания, где содержится **pom.xml**
2. Ввести `mvn clean` -> `mvn compile -X` -> `mvn  package`
3. После компиляции запустить **.jar** файл из папки target с помощью `java -jar ${путь.jar}`

## Задание 2 - cервис обмена файлами

После пуска программы на порте 8080 запускается сервер. 
Для доступа необходимо в браузере перейти по адресу http://localhost:8080.

На сайте есть форма, при нажатии на которую открывается проводник, в котором можно выбрать файл для загрузки.

После выбора необходимо нажать на кнопку **Upload**. В случае успеха снизу появится ссылка на скачивание.

Все сериализованные файлы сохраняются в папке `file-storage`, а их метаданные в `file-metadata.json`.
